## 工程说明

    使用的是Unity 2019 STL 版本，主要是将之前写的一些优化模块进行整合（包括UWA 和部分开源库，整理完以后会将开源库链接置于底部），属于整理个人代码。

### 1.图片查看和批量压缩

    工具说明：
    1.图片所有的优化信息都在TextureOptimization.asset 里，如果使用里Odin 可以直接执行。
